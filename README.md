## 🚀 Featured Projects

### 🛰️ Satellite Change Detection System
> PyTorch · FastAPI · Celery · Redis · PostgreSQL/PostGIS · Sentinel-2 · Leaflet.js

- Trained **ChangeFormer** (EfficientNet-B0 + transformer decoder) on OSCD dataset —
  **IoU: 0.586, F1: 0.736**
- Detected **14.63 km²** of urban change over Bengaluru (2021→2023) on real ISRO imagery
- Built full GIS pipeline: JP2 ingestion → AOI clipping → patch tiling →
  Gaussian stitching → GeoTIFF/GeoJSON output
- Async production system with Copernicus STAC API, 8-year timeseries, and
  threshold-based email alerts

---

### 🌊 Coastal Tide Predictor
> PyTorch · Lightning · FastAPI · Docker · PostgreSQL · React · CUDA

- Trained **Temporal Fusion Transformer** on 8,760 hrs of NOAA tidal data
- Benchmarked **11 models** (SARIMA, XGBoost, LSTM, GRU, TCN, Prophet, Holt-Winters)
  — TFT outperformed all by **65.65% RMSE**
- Best metrics: **RMSE 0.0080 m**, **MAPE 0.96%**
- 32-feature autoregressive input: 6 tidal harmonics, lagged observations,
  rolling stats, cyclical encodings
- React frontend with **15 US coastal stations**
